## Overview

GreenChain is a web application that lets Ethereum users link their public wallet address to automatically calculate the carbon footprint of their on-chain activity. The dashboard surfaces real-time emissions data, cross-chain carbon scores, and offset recommendations — helping users understand the environmental impact of their transactions and make greener choices.

> Built from concept to working demo in under 48 hours as part of the H.E.R. DAO Hackathon, where it placed **1st overall**.

---

## Features

- **Wallet-linked carbon tracking** — paste any Ethereum public address to instantly pull transaction history and calculate total emissions
- **Carbon Emissions chart** — visualises your footprint over time with monthly/weekly toggles
- **Blockchain Score** — benchmarks your wallet's environmental performance across chains (Chia, Bitcoin BSC, Tezos, Solana, Mooky, Chimpzee)
- **Transaction timestamps** — granular breakdown of gas price, electric consumption, and carbon footprint per transaction
- **Blockchain Margins** — net carbon reduction per ten transactions and margin reduction by chain
- **Offset suggestions** — actionable recommendations to reduce your footprint and switch to greener alternatives

## Getting Started

```bash
# Clone the repository
git clone https://github.com/seansqlifting29/GreenChain.git
cd GreenChain

# Install dependencies
npm install

# Start the development server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the app in your browser.

---

## Usage

1. Open the dashboard
2. Enter your Ethereum public wallet address in the account field
3. Your carbon emissions history loads automatically
4. Use the **Month / Week** toggle on the Carbon Emissions chart to adjust the timeframe
5. Review your **Blockchain Score** to see how your activity compares across chains
6. Check the **Blockchain Margins** table for offset strategies
